# Server Room Design Documentation

This repository documents the design, implementation, and best practices
of a small-to-medium enterprise server room.

## Environment Overview
- 2× Dell PowerEdge R220 Servers
- Omada Network Management System
- 2× Core Switches
- MikroTik Router
- 7× ADSL Routers (Multi-WAN)
- Central UPS System
- Structured Cabling & Patch Panels

## Objectives
- High availability
- Clean physical & logical design
- Proper power and cooling
- Enterprise-grade documentation
